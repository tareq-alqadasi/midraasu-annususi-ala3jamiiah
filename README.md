# midraasu-annususi-ala3jamiiah
برنامج يعرض النص الاعجمي مقسم الى الجمل مع ترجمة موازية في العرض و يقدم خدمة الاداء الصوتي القابل للتسريع بهدف دراسة النص الاعجمي و فهم محتواه و لغته مع قدرة على التحكم بسرعة العملية في الحد المعقول  ملاحظة: البرنامج استخدم في تطويره الذكاء الاصطناعي چات جيبيتي لتوفير الوقت - مجموع مدة التطوير : ٩ ساعات -
